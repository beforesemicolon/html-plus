HTML+ engine allows you to specify a Typescript file in your template and it will compile it for you and serve it to the browser.

You can add a tsconfig.json file at the root of your project to control things but no errors will be thrown in the console. For the browser, all typescript errors are ignore but you can setup linters and run tsnode process along with the server's to be notified of errors.

This feature allows you to write future Javascript for the browser and no setup is needed for your projects.
