With HTML+ you can write smaller and more powerful HTML using it as template language and with the power of the express engine, it can manage all pages and resources connected to the pages like CSS, SASS, LESS, Stylus and Typescript so you can write the most modern code you can.

When you are ready to deploy, it comes with a static site builder to build your project for production or you can simply continue to use it in a live server for a dynamic website with all batteries included.

Requirements

  • Node 10 or later
  • Any OS that NodeJs can run
  • Express 4 or later

Create Your Project

You can create your node project as you would normal do or executing the following commands:

#create directory named "my-app" mkdir my-app #navigate inside the created directory cd my-app #initialize npm and follow the prompt npm init

Installation

Inside your newly created project you can proceed to install HTML+ and express:

npm install express npm install @beforesemicolon/html-plus

Server Setup

Create your server file and a pages directory(can be named anything you want).

mkdir pages touch server.js

Your project file structure should look like so:

my-app - pages - server.js - package.json - package-lock.json

Inside your server.js file the simplest setup you can have is importing HTML+ engine, create your express app to pass it to the engine along with the path to the directory where you will put the html page files.

// server.js const express = require('express'); const http = require('http'); const path = require('path'); const {engine} = require('@beforesemicolon/html-plus'); const app = express(); engine(app, path.resolve(__dirname, './pages')); const server = http.createServer(app); server.listen(3000)

HTML+ engine will setup your Routes for your pages and all handlers for CSS(including preprocessors), Javascript and Typescript files.
