The Transform API is how all files are processed internally and this API is exposed to you so you can build things on top of it for more control.

const {transform} = require('@beforesemicolon/html-plus');

This object expose a few methods which are language specific.

Options

Options vary for each but so please refer to the Transform API reference for more details on supporting options for each.

One option they all have in common is the file option which is a way HTML+ is trying to make it easy to work with files in general.

const {transform, File} = require('@beforesemicolon/html-plus'); const path = require('path'); const file = new File(path.resolve(__dirname, 'app.sass')); await transform.sass({file});

The File class is a powerful way to load files and create them and it is used internally for everything as it does the heavy lifting. Refer to its API reference for more details.
