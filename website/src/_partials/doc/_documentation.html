Welcome!

HTML+ was built to simplify your journey in building professionally looking websites without spending any time of deciding which tool to use to build or run your it or even how to optimize it so you can have it running live on a server. HTML+ got you cover there.

With simple API, small learning curve and batteries included, starting to use it takes almost no time.

Key features:

  • Zero configuration needed;
  • Automatic routes configuration;
  • Server Side RenderNode;
  • Powerful templating features;
  • Robust template data contextualization;
  • Custom tags and attributes for reusability and extra power
  • CSS preprocessors, POST CSS and Typescript support out of the box;
  • Optimizes code for production;
  • Fast page rendering for a live websites;
  • Build optimized static websites;
  • Advanced configuration options;

This documentation will provide you with all details you need to start your next project. Please refer to the learn tutorial if you prefer learning by example with a real world project tutorial.
