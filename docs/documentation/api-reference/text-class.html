Text

Text details
Details
name Text
type class
arguments A text string
properties value, type
methods toString
description A class representing HTML comment.

Constructor

The constructor takes the text string and a optional object representing the context data.

Properties

value
type: string: The text value.
type
type: string, readonly: Always returns 'text'.

Methods

toString
type: function: Function that returns an HTML text string.

Usage Examples

const path = require("path");
const {Text} = require("@beforesemicolon/html-plus");

const text = new Text('total is {total}', {
	total: 200
});
console.log(text.toString());
