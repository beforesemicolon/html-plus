HTMLNode

HTMLNode details
Details
name HTMLNode
type class
arguments HTML string, options
properties tagName, innerHTML, rawAttributes, attributes, context, type
methods setAttribute, removeAttribute, setContext, removeContext, clone, duplicate, childNodes, renderChildren, render, toString
description A class representing HTML node tree.

Constructor

The constructor takes a required absolute path to the file and an optional source directory.

Properties

type
type: string, readonly: Always returns 'node'.
tagName
type: string | null, readonly: The name of the tag.
innerHTML
type: string, readonly: The inner content of the tag.
rawAttributes
type: string, readonly: Attribute as they appear on the tag.
attributes
type: object: Key-value pair object literal of the attributes.
context
type: object, readonly: Key-value pair object literal of the context data.

Methods

setAttribute
type: function: A function that takes a key and a value string where the value can contain curly brace that can be bind and saved to be accessed on the attributes object.
removeAttribute
type: function: A function that takes a attribute name and removes it.
setContext
type: function: A function that takes a key string and a value and updates the node current context.
removeContext
type: function: A function that takes a attribute name string and removes from the current context.
clone
type: function: A function that will return a copy of the node and has no reference to the parent node. If the clone has reference to the parent node context data, rendering will result in error.
duplicate
type: function: A function that copy the node in place and returns a copy of the node. This means that the parent child node list will be updated with a new node. This function can also take a object representing the context data for the new copy.
childNodes
type: function: A function that takes an optional data context data object and returns a list of child nodes -- HTMLNode, Text and Comment.
renderChildren
type: function: A function that takes an optional data context data object and returns a string result of rendering all internal nodes.
render
type: function: A function returns a string result of rendering the node.
toString
type: function: Alias to the render function.

Options

HTML options details
Options
data A Javascript object representing the data you want to pass to the markup. Can be referenced to as $data in the markup.
context A Javascript object literal representing the context data the markup can refer to by the keys.
customTags An array of valid function or class based tag.
customAttributes An array of valid Attribute based custom attribute.
rootNode An HTMLNode instance normally used by partial file to grab content.
onTraverse A function that is called on render for every HTMLNode with the node itself and the File instance.
partialFiles An array of PartialFile instance to be used by the markup especially if it contains a include tag.
env The environment, 'development' or 'production'.
file A File instance.

Usage Examples

const path = require("path");
const {HTMLNode} = require("@beforesemicolon/html-plus");

const title = new HTMLNode('<h2>{title}</h2>', {
	context: {title: 'My title'}
});
