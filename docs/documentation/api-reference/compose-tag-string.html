composeTagString()

composeTagString details
Details
name composeTagString
type function
arguments node, content, excludedAttributes
description A function to compose an HTML tag with it's attributes and content.

Arguments

node
type: HTMLNode-like, required: a HTMLNode instance or HTMLNode-like object to compose as tag string.
content
type: string, optional: The HTML string representation to be used as the HTML or text content.
excludedAttributes
type: string[], optional: A list of attribute names to exclude from the HTML tag string.

Return

This function will return an HTML string representation that can be used to render the node, for example.

The content is never parsed. Whatever the content you pass to it will be used as is.

Usage Examples

const {composeTagString, HTMLNode} = require("@beforesemicolon/html-plus");

const node1 = new HTMLNode('<h3>some title</h3>');
const node2 = new HTMLNode('<h3 style="display: block;">some title</h3>');

composeTagString(node, 'different content');
// returns "<h3>different content</h3>"

composeTagString(node, node2.innerHTML, ['style']);
// returns "<h3>some title</h3>"

composeTagString({tagName: 'h1'}, node1.innerHTML);
// returns "<h1>some title</h1>"

This is what the script tag implementation looks like.

const {composeTagString} = require("@beforesemicolon/html-plus");

function Script(node) {
    return () => composeTagString(node, node.innerHTML);
}

module.exports.Script = Script;
